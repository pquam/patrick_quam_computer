Website

This website is hosted on a Akami/Linode VM running Debian.

Specifications:

  • CPU: Shared, 1 Core Reserved
  • RAM: 2GB

Software:

  • NGINX - Hosts web pages, directs HTTP & HTTPS traffic.
  • fail2ban - Blocks connections from unwanted IPs, particularly from botnets.
  • Wazuh Agent - Intrusion prevention / detection & malware scans.
  • CertBot - Generates SSL certificates for HTTPS.
  • git - Manage beta and production versions of the website.
