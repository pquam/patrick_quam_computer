Software

I use Visual Studio Code for development on both my desktop and my laptop. I use Windows on the desktop because I often find myself needing the compatibility with NVIDIA Graphics drivers, and various applications such as Adobe Photoshop and new Steam games. Windows Subsystem for Linux and Windows PowerToys add a ton of functionality to Windows 11 that makes it exceptional for power users, while retaining a clean interface. I use the ARC web browser.

Website
