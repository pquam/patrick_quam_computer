The stack

Welcome to my personal website, a testament to my interest in web development and server management, and a good excuse to build a really cool computer. While I had the assistance of ChatGPT in some areas (like writing this description), the majority of the site was crafted by me using VSCode and VIM. I am utilizing Bootstrap for design and responsiveness. The website you're viewing is hosted on a VPS, which I set up and manage from the ground up. This involved hardening and configuring both the VPS and the server for enhanced security, as well as automating server updates, and ensuring timely SSL certificate renewals using Certbot using bash scripts and cron jobs. Additionally, I maintain a local beta version of this site on a custom server I built myself. Both the live and beta versions are managed seamlessly using Git and GitHub.

What is it for?

This website primarily serves as my digital notebook and a playground for exploring intriguing technologies. Through its creation, I've deepened my knowledge of Linux, automation, server operations, DNS intricacies, and honed my proficiency in HTML, CSS, and JavaScript. My journey with this site has also solidified my grasp on Git and GitHub. As I continually enhance and broaden the site's offerings, I'm eager to delve deeper into JavaScript and other web development tools.
